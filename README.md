# Navya Thellapati Portfolio

An independent personal portfolio website for Navya Thellapati, built with React, TypeScript, Vite, Tailwind CSS, Framer Motion, React Router, and Lucide React.

## Features

- Floating responsive navigation with theme toggle and resume download
- Animated hero with custom backend/API/AI engineering visual
- Impact metrics, engineering focus cards, selected project showcases, experience, bento expertise, education, certifications, and contact CTA
- Dedicated case-study routes for DocuMind, MyChart, and Responsible AI
- Light/dark theme support with persisted preference and reduced-motion support
- Original CSS/React product mockups, no copied screenshots or placeholder links

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

## Deployment

The app is ready for Vercel as a Vite single-page application. `vercel.json` rewrites all routes to `index.html` so case-study URLs work on refresh.
